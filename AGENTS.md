# Agent Notes

This repository uses UTF-8 for all source and documentation files.

When editing files:

- Preserve existing Chinese comments. Do not delete or replace them unless the user explicitly asks.
- Do not introduce mojibake. If Chinese text appears corrupted, pause and repair the encoding or ask before changing the text.
- Write new files as UTF-8.
- Prefer UTF-8-safe editing tools and commands. Avoid PowerShell write paths that may use legacy encodings.
- Keep source comments readable. New comments may be English by default, but Chinese comments are allowed and must remain valid UTF-8.
